# Wiki starter in Django

## How To Setup
```
git clone https://github.com/minotaurdsp/miki.git
```


### Make new env 
```
python3 -m venv venv
```

### Activate env 
```
source venv/bin/activate
```

### Install requirements 
```
pip install -r requirements.txt
```

### Import demo data
```
python manage.py makemigrations
```

### Start app
```
python manage.py makemigrations
python manage.py migrate
```
